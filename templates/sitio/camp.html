DepWebCamp

En este sitio web, una empresa promociona su campamento de conferencias con ponentes expertos en diferentes tecnologías. Para la entrada al evento se tienen diferentes boletos con precios diferentes. También se da información como la ubicación o conferencias de los días del evento.

Aún sigo trabajando para conseguir un host para este sitio web, por lo que por el momento no es posible interactuar con el sitio web, pero te dejo algunas imágenes de su implementación.

Como usuario puedes registrarte para comprar un pase, ya sea gratis o el que ofrece una experiencia más grande.

Como administradores del evento se cuenta con un panel donde podemos realizar varias acciones, como ver a las personas registradas, la información de los ponentes, que impartirán las conferencias, así como añadir más o eliminarlos en caso de que se haya cancelado su participación. También podemos agregar nuevos eventos, evento y asignarlos a un ponente. 

Para que un usuario pueda comprar una entrada necesita pagar, y como pruebas realice pagos usando la API de PayPal.
