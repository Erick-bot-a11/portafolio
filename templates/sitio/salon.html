Salón de belleza

Como cliente del negocio de un salón de belleza puedes agendar citas para mayor flexibilidad, es necesario crear una cuenta en este sitio web, si ya la tienes pues puedes iniciar sesión, o si olvidaste tu contraseña puedes restablecerla. Todas estas funciones hacen consultas a una base de datos para verificar que los datos sean correctos, además uso MailTrap para enviar correos a mi mismo y crear cuentas como testing, así que en el entorno de producción aún no están disponibles todas estas funciones. Sin embargo puedes iniciar sesión con una cuenta de demostración, y probar las funciones del sistema.

Debido a otros factores, al entrar al sitio web puede marcarlo como peligroso, pero te aseguro que es totalmente seguro.

Usuario de Ejemplo: demostracion@correo.com

Contraseña: 222222

Ver Sitio Web

Como usuario puedes seleccionar servicios, (que son dinamicos) seleccionar una hora de la cita y confirmar la reserva.

Si eres un administrador del sistema (dueño del salón de belleza) puedes ver las citas programadas para una fecha, ver los servicios que tienes disponibles y si lo requieres editar el precio o el nombre, así como eliminarlo. También puedes agregar nuevos servicios disponibles.
